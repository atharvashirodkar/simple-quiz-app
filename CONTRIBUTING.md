## üõ† How to Contribute

### 1. Fork the repository
Click the **Fork** button on the top right of this repo page to create your own copy.

### 2. Clone your fork
```bash
git clone https://github.com/atharvashirodkar/quiz-app.git
cd quiz-app
```

### 3. Create a new branch
Always work on a separate branch for your feature or fix:
```bash
git checkout -b feature/your-feature-name
```

### 4. Make your changes
- Keep your code clean and well-structured.
- Use meaningful commit messages.
- Follow existing naming conventions in code and folders.

### 5. Test your changes
Before submitting, make sure:
- The quiz works correctly in your browser.
- Buttons, answers, and score logic function as expected.
- No console errors appear in Developer Tools.

### 6. Commit and push
```bash
git add .
git commit -m "Add: Short description of what you changed"
git push origin feature/your-feature-name
```

### 7. Submit a pull request (PR)
- Go to your forked repository on GitHub.
- Click **Compare & pull request**.
- Add a clear title and description.
- Wait for review/feedback before merging.
